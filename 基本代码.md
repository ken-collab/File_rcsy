# 基本代码

**一次请创建一个rcsy文件**

## 初始化

```python

```

filename是文件名

mode是python文件操作类型

调用就可以得到一个空的rcsy文件

## 写入

### 基本写入

```python
def write_rcsy(name, text):
```

该种写入为所有关系（像变量一样），中间用**--**连接

例如：

```python
import rcsy as r
r.into("MyRcsy.rcsy", "w")
r.write_rcsy("My_name", "KenZhu")
```

执行后会得到一个rcsy文件，里面写着：

```rcsy
My_name--KenZhu
```

**不要在--前后加空格**
